## How to Run

1. Make sure you have a C++ compiler installed (e.g., `g++`).
2. Navigate to the relevant folder:
   ```bash
   cd 01-basics
   ```
3. Compile a file:
   ```bash
   g++ filename.cpp -o output
   ```
4. Run the executable:
   ```bash
   ./output        # Linux/macOS
   output.exe      # Windows
   ```
